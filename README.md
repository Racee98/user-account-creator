# Linux User Account Creator

A simple Bash script that helps Linux system administrators create new users quickly and securely.

## 🚀 Features
- Creates a user with a home directory
- Prompts for full name (stored as user comment)
- Sets a password interactively
- Forces password change on first login
- Basic error handling and user feedback

## 🧰 How to Use
Run as root or with sudo:
```bash
sudo ./create_user.sh
